# Projeto Shopping Cart!

Nesse projeto foi criado uma aplicação utilizando a API do [Mercado Livre](https://developers.mercadolivre.com.br/pt_br/itens-e-buscas), HTML, CSS, JavaScript e LocalStorage.

## Desempenho:
